# Завдання 7. Метод Монте-Карло: два кубики

## Мета
Змоделювати велику кількість кидків двох чесних кубиків, порахувати ймовірність кожної суми (2–12) методом Монте-Карло та порівняти з аналітичними значеннями.

## Порівняння результатів
Програма виводить таблицю:
- ймовірність Монте-Карло для кожної суми
- аналітичну ймовірність
- різницю між двома методами

## Висновки
1. Оцінки Монте-Карло наближаються до аналітичних ймовірностей при великій кількості симуляцій
2. Найбільшу ймовірність має сума 7, що підтверджується симуляцією
3. Різниця між Monte Carlo та аналітикою зменшується зі зростанням `n_rolls`, що підтверджує коректність методу Монте-Карло для цієї задачі.
